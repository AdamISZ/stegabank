Welcome to Alpha testing of Paysty ['peisti].

A decentralized and fully open-source fiat (e.g. USD,EUR,JPY etc.) to BTC exchange.

You should have received a private key from waxwing in order to use this software. Please save the key in alphatest.txt in the installation directory (i.e. the directory from which you started Paysty). After that the fun can be started.
